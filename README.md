# Links

A modern, Linktree-style link hub built with React, designed for showcasing multiple links in one place.

[![Live Demo](https://img.shields.io/badge/🌐_Live-Demo-2ea44f?style=for-the-badge)](https://cloud.umami.is/q/70EdrgyuB)

## Features

- **Modern UI** - Clean design built with React and Tailwind CSS
- **Multi-language Support** - Internationalization with i18next
- **Form Handling** - Contact form powered by [Formspark](https://formspark.io/)

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, Lucide React icons
- **Internationalization**: i18next
- **Form**: [Formspark](https://formspark.io/)
